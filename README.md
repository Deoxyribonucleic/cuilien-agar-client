pyAgar.io
=========

agar.io Python client

`python3 window.py`

Requires `websocket-client` module and GTK.

Controls
--------
| Key       | Action                |
|:----------|:----------------------|
| `R`       | respawn               |
| `W`       | shoot small cell      |
| `Space`   | split                 |
| `Mouse`   | movement direction    |
| `H`       | show/hide debug hints |
| `Q`/`ESC` | quit                  |
